Track, find, and manage all shortcodes across your WordPress site in a user-friendly table. Filter by shortcode type, post type, and post status to quickly locate and review shortcode usage.

== Description ==

**Shortcode Tracker** is a powerful and user-friendly tool for managing shortcodes across all your WordPress posts and pages. With Shortcode Tracker, you can locate specific shortcodes, see where they’re used, and get an overview of their usage in a WordPress-styled table. 

**Features:**
- Search for specific shortcodes across your entire site
- Filter results by post type (e.g., post, page) and post status (Published, Draft, Private, Trash)
- View shortcode usage details and get direct links to posts or pages containing them
- Export results as a CSV for easy record-keeping

This plugin is ideal for site administrators, content managers, and developers who need to ensure accurate and organized shortcode usage across a site. 

== Installation ==

1. Download and unzip the plugin file.
2. Upload the `shortcode-tracker` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the "Plugins" menu in WordPress.
4. Go to **Tools > Shortcode Tracker** to start using the plugin.

== Frequently Asked Questions ==

= How do I search for a shortcode? =
After activating the plugin, go to **Tools > Shortcode Tracker**. Select your desired shortcode from the dropdown menu, along with optional filters for post type and status, then click "Find Shortcode."

= Can I export shortcode data? =
Yes, you can export the shortcode usage data as a CSV file, making it easy to keep a record of all occurrences.

= Does this plugin support custom post types? =
Yes, you can filter results by any public post type on your site.

= Is there a way to see the shortcode content? =
The plugin displays each shortcode instance within its post or page, so you can quickly see its context.
